# To-Do List Application

A clean, lightweight, terminal-based To-Do List utility built using Python. This application allows you to create, track, update, and delete tasks seamlessly, using a local JSON file to ensure your progress is saved automatically between sessions. Live deployed at : https://dapper-salmiakki-1bd020.netlify.app/

## 🚀 Features

* **Create Tasks:** Instantly append new items to your list.
* **Live Tracking:** Displays your task backlog with intuitive status indicators (`[ ]` for pending, `[✓]` for completed).
* **Update Status:** Toggle tasks as complete using their listed index numbers.
* **Delete Tasks:** Remove accidental or stale entries on the fly.
* **Persistent Storage:** Saves data locally to `todo_list.json` automatically after every action.
